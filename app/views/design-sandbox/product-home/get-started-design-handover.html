What we've changed

We have changed the priority of information and content structure on the Get started screen so all RP users have a clear call to action to create an admin account. This supports the changes to the Config controls, where users can decide if they need the identity verification feature and supports the CDDO mandate for all government services using GOV.UK One Login, rather than it being a choice.

This is a high level description of the content iteration:

  • removed the numbered H2/li>
  • Changed the paragraph content to standard body text, not bold
  • Removed 'You can' from link text sentence
  • Removed numbered list format from H2
  • Changed sentence structure to front load it with the action we want users to take next
  • Added an H3 to support scan reading/navigation by headings
  • Removed "if" clause in sentence and restructured sentence to front load using active, action-linked phrasing
  • Reformatted second sentece into a bullet list for scannability
  • changed position of sign in link to communicate hierarchy of actions
